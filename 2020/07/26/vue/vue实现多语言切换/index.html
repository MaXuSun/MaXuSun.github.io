vue实现多语言切换
发表于 2020-07-26| 更新于 2021-11-21|Vue
阅读量:

今天做了一个项目，需要用户点击语言切换时，能够进行中英文的语言切换。后来发现需要使用vue-i18n

  1. 安装vue-i18n
    直接在命令行输入npm install vue-i18n --save-dev。或者在package.json文件的devDependencies添加"vue-i18n": "^8.18.2",。然后运行npm install

  2. 在src/components下新建一个文件夹lang，并在lang下新建zh.js和en.js文件。然后将自己需要切换的文字都对应写在这两个文件里。q其中的language,user,signin等可以根据自己要求命名，格式是这样的就行。

  3. 在main.js下引入及注册vue-i18n

  4. 在dom里使用$t('')，若在js里使用则是this.$t('')
